ИНФОРМАЦИОННАЯ СИСТЕМА УЧЕТА СКЛАДСКИХ ТОВАРОВ

1. ОБЩЕЕ ОПИСАНИЕ
   Данный проект представляет собой консольное приложение на языке Python 3.13, предназначенное для учета складских товаров.
   Система позволяет управлять товарами, складами, поставщиками и остатками на складе.

В качестве базы данных используется SQLite, что позволяет хранить данные в одном файле без установки дополнительных СУБД.

---

2. СТРУКТУРА ПРОЕКТА

Проект состоит из следующих файлов и папок:

config.py
Содержит имя базы данных.

main.py
Главный файл запуска программы.

database/db_manager.py
Модуль для создания базы данных и подключения к ней.

models/
Содержит классы моделей:

* product.py – работа с товарами
* warehouse.py – работа со складами
* stock.py – учет остатков
* supplier.py – поставщики
* category.py – категории товаров

ui/
Содержит пользовательский интерфейс:

* menu.py – главное меню
* menu_products.py – управление товарами
* menu_warehouses.py – управление складами

---

3. ФУНКЦИОНАЛ СИСТЕМЫ

Система позволяет:

* добавлять товары

* редактировать товары

* удалять товары

* просматривать список товаров

* добавлять склады

* просматривать склады

* учитывать количество товаров на складе

---

4. БАЗА ДАННЫХ

В системе используются следующие таблицы:

categories – категории товаров
suppliers – поставщики
warehouses – склады
products – товары
stocks – остатки товаров на складах

Связи между таблицами реализованы через внешние ключи (FOREIGN KEY).

---

5. ПРИНЦИП РАБОТЫ

При запуске программы выполняется функция initialize_db(), которая:

* создает таблицы, если они отсутствуют
* подготавливает базу данных к работе

Далее пользователю отображается главное меню, через которое он может управлять системой.

---

6. ТРЕБОВАНИЯ

Для запуска программы необходимо:

* Python 3.10+ (рекомендуется 3.13)
* стандартные библиотеки Python (sqlite3)

---

7. ЗАПУСК ПРОГРАММЫ

Запуск осуществляется командой:

python main.py

---

8. ДОПОЛНИТЕЛЬНАЯ ИНФОРМАЦИЯ

Система может быть расширена:

* добавлением отчетов
* учетом движения товаров
* графическим интерфейсом
